✨ Features
🧘 Verse of the Day: Start your day with a new, inspirational verse featured on the homepage.
😊 Mood-Based Exploration: Select your current mood (e.g., Motivated, Stressed, Confused) to receive a curated list of verses that offer guidance and comfort.
🔍 Powerful Search: Instantly find specific verses or themes by searching with keywords like "duty," "peace," or "soul."
❤️ Personal Favorites: Save verses that resonate with you. Your collection of favorite verses is always just a click away for reflection.
🛠️ Built With
GitaMind is a full-stack application built with the MERN stack:
MongoDB: A NoSQL database used to store the collection of Bhagavad Gita verses, translations, and user data.
Express.js: A backend web application framework for Node.js, used to build a robust RESTful API.
React.js: A frontend JavaScript library for building a fast, interactive, and dynamic user interface.
Node.js: A JavaScript runtime environment that executes the backend server-side code.

You must have the following software installed on your machine:
Node.js (v14 or newer)
